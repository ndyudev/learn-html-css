06. Forms & Inputs - Biểu Mẫu & Đầu Vào

🎯 Mục Tiêu Học Tập

  • Hiểu cách sử dụng thẻ <form> để tạo biểu mẫu
  • Biết các loại input khác nhau: text, email, password, checkbox, radio
  • Hiểu cách sử dụng <label>, <fieldset>, <legend>
  • Thực hành tạo form đăng ký và đăng nhập hoàn chỉnh

📚 Lý Thuyết

1. Form Cơ Bản (<form>)

Form được sử dụng để thu thập dữ liệu từ người dùng.

📝 Cú Pháp Cơ Bản

<form action="/submit" method="post">
    <label for="username">Tên đăng nhập:</label>
    <input type="text" id="username" name="username">
    
    <label for="password">Mật khẩu:</label>
    <input type="password" id="password" name="password">
    
    <button type="submit">Đăng nhập</button>
</form>

2. Các Loại Input Cơ Bản

📝 Text Input

<input type="text" name="fullname" placeholder="Nhập họ tên">

📧 Email Input

<input type="email" name="email" placeholder="example@email.com">

🔒 Password Input

<input type="password" name="password" placeholder="Nhập mật khẩu">

3. Checkbox & Radio Buttons

☑️ Checkbox (Chọn Nhiều)

<input type="checkbox" id="hobby1" name="hobbies" value="reading">
<label for="hobby1">Đọc sách</label>

🔘 Radio Buttons (Chọn Một)

<input type="radio" id="male" name="gender" value="male">
<label for="male">Nam</label>

4. Select & Textarea

📋 Select Dropdown

<select name="country">
    <option value="">Chọn quốc gia</option>
    <option value="vn">Việt Nam</option>
</select>

📝 Textarea

<textarea name="message" rows="4" cols="50" 
    placeholder="Nhập tin nhắn của bạn..."></textarea>

5. Form Validation & Attributes

✅ Required Fields

<input type="text" name="username" required>

📏 Min/Max Length

<input type="text" name="username" minlength="3" maxlength="20">

💻 Thực Hành

Bài Tập: Tạo Form Đăng Ký

Tạo form đăng ký tài khoản với đầy đủ validation

Code HTML:

<form action="/register" method="post">
    <fieldset>
        <legend>Thông Tin Cá Nhân</legend>
        
        <div>
            <label for="fullname">Họ và tên:</label>
            <input type="text" id="fullname" name="fullname" required>
        </div>
        
        <div>
            <label for="email">Email:</label>
            <input type="email" id="email" name="email" required>
        </div>
    </fieldset>
    
    <button type="submit">Đăng Ký</button>
</form>

🧠 Kiểm Tra Kiến Thức

Câu Hỏi 1:

Thuộc tính nào được sử dụng để bắt buộc người dùng nhập dữ liệu?

Câu Hỏi 2:

Thẻ nào được sử dụng để nhóm các trường liên quan trong form?

📝 Tóm Tắt

  • <form> tạo biểu mẫu để thu thập dữ liệu
  • HTML5 cung cấp nhiều loại input: text, email, password, number, date
  • Checkbox cho phép chọn nhiều, radio chỉ chọn một
  • Fieldset nhóm các trường liên quan
  • Validation attributes: required, minlength, maxlength, pattern
  • Luôn sử dụng label cho accessibility
